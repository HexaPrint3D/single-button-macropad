# single-button-macropad
I built a single-button macropad!

### requirements

1.Hardware
- esp32s3 devkit with built in neopixel led
- transparent machanical keyboard switch 15.5 x 15

2.Tools:
- 3d Printer for case
- soldering iron

files for 3d printing: https://www.printables.com/model/1773255-single-button-macropad
